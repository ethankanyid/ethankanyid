Other Notes

Steam

Go into settings and and compatibility. Enable steam support for all other devices. Go to interface and select start menu, run steam when start, and big picture mode if you prefer. Also disable notifications for upcoming games. Go into friends and don't start friends when steam runs.
As far as controller support. use the 2.4 ghz dongle. and for each game, enable steam input translation. May or may not use nintendo button layout in controller in settings

Note: Ui is messed up so try turning on and off ui scaling in interface.

Modding

You're going to want to use bottles, lutris, playonlinux, steamtinkerlaunch. As well, flatseal will help you manage permissions. This tutorial was helpful

Pacman

Gives access to pacman and its repositories

Install: sudo bash -c "$(curl -fsSL https://pacstall.dev/q/install || wget -q https://pacstall.dev/q/install -O -)"
Uninstall: sudo bash -c "$(curl -fsSL https://pacstall.dev/q/uninstall || wget -q https://pacstall.dev/q/uninstall -O -)"

Wine

Install wine from the terminal, “sudo apt install wine”. run “winecfg”. Then, install exe for windows application and run using “wine FILENAME” and make sure your in the right directory.

Newest Drivers

sudo apt install nvidia-driver-XXX
sudo apt remove nvidia-driver-YYY
sudo apt autoremove

Upgrading Mint Version

Install and run mint upgrade. Use time shift to create a back up. update software, drivers,…

Timeshift

Timeshift is used for backups. I used rsync and I stored the backup on my root partition. This won't back up the home partition, and don't let it unless you have incredible space. It's used to restore the snapshot of an os in case of failure. Ideally your home partition shouldn't be compromised because it's partitioned.
You can use backup tool for your home folder
