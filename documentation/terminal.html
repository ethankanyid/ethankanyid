ZSH

This is a helpful tutorial for customizing your terminal. ohmyzsh can be used on any terminal running zsh. On mac, iterm2 is an alternate terminal.

So, if you havent already, make sure you have a few things:

  • Homebrew - if on mac
  • Iterm2 - if on mac
  • ZSH

Then you'll be needing:

  • iterm2 color pallete- if on mac
  • ohmyzsh
  • powerlevel10k

You'll need to edit your ~/.zshrc and display changes by running "source ~/.zshrc"

Git

From the terminal, move to the directory you want to create a repository in.
Then these commands will then be of value:

git init
initialize repository
git status
status of branch/commits/adds/...
git add . (or list files)
stages changes to be committed
git commit -m “message”
commits the changes with custom message
git log
see changes
git checkout
see previous commit or branch
git branch
shows branches or add name for new branch
git merge branchname
merges specified branch to active branch
git remote add origin https://github/...
origin is name for the remote
git remote remove name
removes remote
git push -u origin main
pushes main to origin
git remote -v
lists remotes
git pull origin main
pulls changes from main branch at origin
git clone url
clones a repo locally - do not clone a repo into a different repo!

Fork takes a repo and creates a repo for you to edit. A pull request merges it to the forked branch if allowed.

WSL2

Allows you to access your windows system through a linux terminal.

  1. wsl.exe —install
  2. wsl —list —online
  3. wsl —install -d specificdistro
  4. wsl -d specificdistro
  5. sudo apt install build-essentials
