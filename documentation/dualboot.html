BIOS Tweaks

Before you can boot your PC into an OS, you will want to tweak your bios. This is also a good test for PC builders to make sure they have installed and assembled everything correctly. CSM uses outdated mbr, whereas BIOS uses gpt.

  • Disable secure boot
  • Disable CSM in Bios (Legacy Bios)
  • Optional: enable SVM for virtual machines

Eject gparted USB

Installing Windows

Boot windows USB.
Install windows to partition making sure it has not formatted to the rest of the disk or overwrote any unallocated space. I think you can delete the NTFS partition and then use the “new” and it should show you how it is going to partition the drive. If you see that it has overwrote a partition, you need to boot back into gparted.
When windows asks you to sign into microsoft, you can refuse somehow.
Eject windows USB.

Installing Linux

Boot Linux USB.
Use custom install of linux. Find EXT4 partition and delete it. Select unallocated space and add partition. Use primary partition, EXT4, and select mount point as root “/“. Set the size somewhere around 100 Gb.
Add another partition and use the remaining unallocated space. Make it a primary partition, EXT4, and set mount point to “/home”. At the bottom, install efi or boot loader - whatever is the boot manager portion - select it and set it the efi, system reserved portion of the windows installation (should be around a 100 Mb partition, might also say “Windows Boot Manager”).
Install and Eject Linux USB
Boot into Bios and assign the preferred boot order.

Simple Set Up

The shared partition will allow you to pass data back and forth between windows and linux.

Download all your updates and drivers on both systems. Update to the newest kernel on linux mint by going to update manager, view, and linux kernel.
Customize display settings: Set quality and refresh rate.
Dont forget adblockers, preferred browsers, enabling firewalls, customizing privacy and security options, and customizing desktops.
Note: Most OS's both have fairly strong antivirus software already and it has been suggested that downloading a third-party antivirus for them might make them weaker. For additional network security, check out Safing PortMaster it is an open source firewall and SPN (which is like a VPN)

If you want automatic updates, go to software manager and go to edit and the preferences. click the automation tab and select what you like.
